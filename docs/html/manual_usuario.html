Manual de Usuario

Introducción

Este sistema permite descargar de forma masiva comprobantes electrónicos (Facturas, Boletas, Notas) y Guías de Remisión desde Odoo. Los archivos se organizan automáticamente en carpetas por Año, Mes y Tipo de Documento.

Configuración Inicial

Antes de ejecutar cualquier script, asegúrese de tener configurado su archivo de credenciales:

  1. Copie el archivo .env.example y renombrelo a .env.produccion.
  2. Abra el archivo y complete sus credenciales de Odoo: 
    ODOO_URL="https://tu-instancia.odoo.com"
ODOO_DB="tu_db"
ODOO_USER="tu_usuario"
ODOO_PASSWORD="tu_password"

Descarga de Comprobantes

Para descargar Facturas, Boletas y Notas de Crédito/Débito:

python run/descargar_comprobantes.py
Nota: Este script utiliza XML-RPC, es muy rápido y descarga PDF, XML y CDR simultáneamente.

Descarga de Guías

Las guías de remisión requieren un proceso especial que incluye Web Scraping para el PDF:

python run/descargar_guias.py

El proceso es:

  1. Descarga automática de XML y CDR vía API.
  2. Apertura automática de navegador para descargar el PDF "e-Guía AGR".

Reparación de Faltantes

Si alguna descarga falló por problemas de conexión, puede ejecutar el script de reparación:

python run/reparar_faltantes.py

Este script detectará qué documentos no tienen PDF y tratará de descargarlos nuevamente.
