🗓️ 28 de Enero, 2026: Reorganización Modular

Se realizó una reestructuración profunda del proyecto para mejorar la mantenibilidad y prepararlo para un repositorio profesional en GitHub.

🛠️ Cambios Realizados:

  • Estructura Core: Centralización de configuración en core/config.py.
  • Unificación: Integración de descarga de XML y PDF de Guías en un solo flujo.
  • Documentación: Generación de portal de documentación en HTML profesional.
  • Seguridad: Implementación de .env.example para proteger credenciales.

Implementación de documentación en formato Markdown para la Wiki del proyecto en GitHub.

🛠️ Cambios Realizados:

  • Nueva Estructura: Creación de la carpeta docs/wiki/ con archivos .md.
  • Guía de Wiki: Manual paso a paso para la carga y mantenimiento de la documentación en GitHub.
  • Consistencia: Sincronización de contenidos entre el portal HTML y la Wiki MD.
