Resumen del Sistema

Este sistema permite automatizar la extracción de documentos fiscales de Odoo (Facturas, Boletas, Guías de Remisión) para su almacenamiento y auditoría externa. Combina tecnología XML-RPC para datos estructurados y Web Scraping (Selenium) para representaciones gráficas (PDFs).
